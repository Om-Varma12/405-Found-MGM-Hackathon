## SIMULATION LOGIC (Auto-running)

On mount, start simulation:

1. Initialize 9 CCTVs with locations:
   S1: "North Gate"
   S2: "Park Junction"  
   S3: "Mall Cross"
   S4: "East Entry"
   S5: "West Hub"
   S6: "City Center"
   S7: "Market Square"
   S8: "Stadium Road"
   S9: "South Gate"

2. Every 20-40 seconds (random):
   - Pick random CCTV
   - Change status: NORMAL → HIGH TRAFFIC
   - After 15s: return to NORMAL
   - Log event

3. Emergency trigger (2 ways):
   a. Manual: "🚑 TRIGGER EMERGENCY" button
   b. Auto: Every 90 seconds randomly

4. On Emergency trigger:
   - Pick random CCTV (not a hospital)
   - Set status: EMERGENCY
   - Set ambulance_detected: true
   - Generate:
     - confidence score: 0.91-0.97 (random)
     - route: array of 3-5 CCTV IDs toward hospital
     - ETA: 90-180 seconds (countdown)
     - signals_cleared: number of signals on route
   - Move panel to priority position (top-left)
   - Dim all other panels
   - Start ETA countdown
   - When ETA hits 0: "AMBULANCE ARRIVED" event
   - Return all panels to normal after 10s

5. Auto-fine generation:
   - During emergency, every 15s:
     - 40% chance to generate a violation
     - Random plate number (MH format: MH12AB1234)
     - Random intersection from route
     - Add to fine log
